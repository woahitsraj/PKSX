# Bl1ndBeholder Pokemon Saves Fixtures

These Save File fixtures are public third-party gameplay saves used for PKHeX Engine runtime smoke tests.

- Source repository: https://github.com/Bl1ndBeholder/pokemon-saves
- Source commit: `234cf954b1eba905448996499ac862178b697911`
- License: MIT, per the source repository's `LICENSE`

These fixtures are committed as public gameplay saves. They are not sanitized or synthetic fixtures, so tests should avoid implying that trainer or save metadata is private-data-free.
Some extensionless source files are stored with a `.sav` suffix in this fixture tree so browser smoke tests can fetch raw bytes through Vite.
