# PKSX Pokemon Sprites

These files are saved from the PokemonDB sprite gallery for offline use in PKSX.

PokemonDB asks projects not to hotlink its image URLs and states that projects may save the images and upload them to their own hosting instead. PKSX follows that guidance by packaging the downloaded images as local static app assets and never linking to PokemonDB at runtime.

Source gallery: https://pokemondb.net/sprites

The generated `catalog.json` records the PokemonDB source URL and retrieval date for each committed image.
